EtherId is Ethereum Name Registar

The main facts about the EtherId:

  • You can register a Domain name for your Ethereum account for some period of time. The maximum time is 2,000,000 Ehtereum blocks ( about a year )
  • At any moment while you own a domain you can prolong the ownershp.
  • If you failed to prolong the ownership, after expiration time anyoneone can calim the ownership for your domain.
  • You can register as many as you want ID's for the domain you own.
  • If you want to sell your domain, put a price on it. Once the price is set, anyone can pay and take the ownership. No communication with you is needed.

You need to install one of the official Ethereum clients official Ethereum clients

EtherId is an open source project. The code is available from https://github.com/lexansoft/etherid.org
